# 公式与解题模板

这份文档是考前查公式用的。建议打印或单独二刷。

## 一、逻辑

量词否定：

```text
not forall x P(x) = exists x not P(x)
not exists x P(x) = forall x not P(x)
```

唯一存在：

```text
exists x (P(x) and forall y (P(y) -> y=x))
```

蕴涵：

```text
P -> Q = not P or Q
not(P -> Q) = P and not Q
```

德摩根：

```text
not(P and Q) = not P or not Q
not(P or Q) = not P and not Q
```

与非：

```text
not P = P ↑ P
P and Q = (P ↑ Q) ↑ (P ↑ Q)
P or Q = (P ↑ P) ↑ (Q ↑ Q)
P -> Q = P ↑ (Q ↑ Q)
```

或非：

```text
not P = P ↓ P
P or Q = (P ↓ Q) ↓ (P ↓ Q)
P and Q = (P ↓ P) ↓ (Q ↓ Q)
```

## 二、集合与关系

幂集：

```text
|P(A)| = 2^|A|
```

二元关系个数：

```text
A 到 B：2^(|A||B|)
A 上：2^(|A|^2)
```

等价关系：

```text
自反 + 对称 + 传递
```

偏序关系：

```text
自反 + 反对称 + 传递
```

划分对应等价关系：

```text
pi = {A1,A2,...,Ak}
R = union Ai x Ai
|R| = sum |Ai|^2
```

Bell 数：

```text
B1=1, B2=2, B3=5, B4=15
```

传递闭包：

```text
有 a->b, b->c，则补 a->c，直到不能再补
```

## 三、函数计数

设 `|A|=m, |B|=n`。

全部函数：

```text
n^m
```

单射：

```text
m <= n
P(n,m) = n!/(n-m)!
```

双射：

```text
m = n
n!
```

满射：

```text
m >= n
sum_{k=0}^n (-1)^k C(n,k)(n-k)^m
```

## 四、排列组合

排列：

```text
P(n,m)=n!/(n-m)!
```

组合：

```text
C(n,m)=n!/(m!(n-m)!)
```

圆排列：

```text
(n-1)!
```

多重集合排列：

```text
n!/(n1!n2!...nk!)
```

非负整数解：

```text
x1+...+xk=n, xi>=0
C(n+k-1,k-1)
```

正整数解：

```text
x1+...+xk=n, xi>=1
C(n-1,k-1)
```

错排：

```text
D_n = n! * sum_{k=0}^n (-1)^k/k!
D4=9, D5=44
```

容斥：

```text
|A union B| = |A| + |B| - |A intersect B|
不出现 A 和 B = 总数 - |A| - |B| + |A intersect B|
```

## 五、母函数

普通母函数：

```text
G(x)=sum a_n x^n
1/(1-x)=1+x+x^2+...
1/(1-x)^m=sum C(m+k-1,k)x^k
```

指数型母函数：

```text
G_e(x)=sum a_n x^n/n!
```

不限制：

```text
e^x
```

偶数次：

```text
(e^x + e^(-x))/2
```

奇数次：

```text
(e^x - e^(-x))/2
```

至少一次：

```text
e^x - 1
```

二项式：

```text
(1+ax)^n 中 x^k 系数 = C(n,k)a^k
```

广义二项式：

```text
1/(1-x)^m 中 x^k 系数 = C(m+k-1,k)
```

## 六、递推关系

一阶：

```text
a_n = c a_(n-1) + d
```

齐次通解：

```text
A c^n
```

若 d 为常数且 `c != 1`，设常数特解 K。

若 `c = 1`，设一次特解 `Bn`。

高阶常系数：

```text
a_n = c1 a_(n-1) + c2 a_(n-2) + ...
```

写特征方程，求根，再写通解。

## 七、图论

握手定理：

```text
sum d(v) = 2e
```

树：

```text
e = n-1
sum d(v) = 2(n-1)
```

二叉树：

```text
n2 = n0 - 1
```

平面图欧拉公式：

```text
v - e + f = 2
```

每个面度数为 3：

```text
e = 3v - 6
```

每个面度数至少为 4：

```text
e <= 2v - 4
```

邻接矩阵：

```text
n 个顶点 -> n x n 矩阵
```

生成树：

```text
生成树边数 = v-1
破圈需删边数 = e-v+1
```

二部图：

```text
无奇圈 <=> 二部图 <=> 可 2 着色
```

圈图边色数：

```text
偶圈 = 2
奇圈 = 3
```

## 八、常见秒杀题

`A={1,2,3}` 上二元关系个数：`2^9=512`。

`A={1,2,3,4}` 上等价关系个数：`15`。

`K_m,n` 着色数：`2`。

无奇圈连通图着色数：`2`。

n 个顶点树的平均度 D：

```text
D = 2(n-1)/n
n = 2/(2-D)
```

连通平面图 n 个顶点 f 个面，边数：

```text
e = n + f - 2
```

正整数 m>=1，`1/(1-x)^m` 展开系数：

```text
x^k 系数 = C(m+k-1,k)
```
