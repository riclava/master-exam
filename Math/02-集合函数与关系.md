# 集合、函数与关系

这份文档覆盖集合运算、幂集、函数计数、二元关系、关系性质、等价关系、偏序关系、闭包和关系复合。

## 一、集合与幂集

幂集 `P(A)` 是 A 的所有子集构成的集合。

若 `|A| = n`：

```text
|P(A)| = 2^n
```

例：

```text
A = {empty}
P(A) = {empty, {empty}}
P(P(A)) = {empty, {empty}, {{empty}}, {empty, {empty}}}
```

易错点：`empty` 和 `{empty}` 不是一回事。

## 二、集合运算

基本运算：

```text
A - B = {x | x in A and x notin B}
A △ B = (A - B) union (B - A)
```

对称差常用等价式：

```text
A △ B = (A union B) - (A intersect B)
```

证明集合恒等式常用元素法。

例：证明 `(A-B)-C = (A-C)-(B-C)`：

左边表示 `x in A, x notin B, x notin C`。

右边表示 `x in A, x notin C, and not(x in B and x notin C)`。由于已经有 `x notin C`，右边等价于 `x in A, x notin C, x notin B`。两边相同。

## 三、子集个数

n 个元素集合：

```text
子集总数 = 2^n
奇数个元素子集数 = 2^(n-1)
偶数个元素子集数 = 2^(n-1)
```

例：

```text
|A| = 100，子集总数 2^100，奇数元素子集数 2^99。
```

## 四、函数计数

设 `|A| = m, |B| = n`，函数从 A 到 B。

全部函数：

```text
n^m
```

单射存在条件：

```text
m <= n
```

单射个数：

```text
P(n,m) = n!/(n-m)! = n(n-1)...(n-m+1)
```

双射存在条件：

```text
m = n
```

双射个数：

```text
n! 或 m!
```

满射存在条件：

```text
m >= n
```

满射个数：

```text
sum_{k=0}^n (-1)^k C(n,k)(n-k)^m
```

也可写作：

```text
n! * S(m,n)
```

其中 `S(m,n)` 是第二类 Stirling 数。

注意：满射个数不是简单的 `n*m`，遇到这种说法要按容斥公式修正。

## 五、二元关系个数

A 到 B 的二元关系是 `A x B` 的任意子集。

若 `|A| = m, |B| = n`：

```text
关系个数 = 2^(mn)
```

A 上二元关系：

```text
2^(n^2)
```

例：

```text
A = {1,2,3}
|A x A| = 9
A 上二元关系个数 = 2^9 = 512
```

## 六、关系性质

设 R 是 A 上的二元关系。

| 性质 | 定义 |
|---|---|
| 自反 | 对任意 a in A，有 `(a,a) in R` |
| 反自反 | 对任意 a in A，有 `(a,a) notin R` |
| 对称 | 若 `(a,b) in R`，则 `(b,a) in R` |
| 反对称 | 若 `(a,b) in R` 且 `(b,a) in R`，则 `a=b` |
| 传递 | 若 `(a,b) in R` 且 `(b,c) in R`，则 `(a,c) in R` |

既对称又反对称的关系：除对角线外不能有任何双向或单向的非对角元素；如果题目要求同时对称和反对称，则非对角线全为 0，对角线可选 0 或 1。

若 `|A| = n`，既对称又反对称的关系个数为：

```text
2^n
```

## 七、等价关系

等价关系满足：

```text
自反 + 对称 + 传递
```

等价关系与集合划分一一对应。

若 A 的一个划分为：

```text
pi = {A1, A2, ..., Ak}
```

对应等价关系：

```text
R = (A1 x A1) union (A2 x A2) union ... union (Ak x Ak)
```

有序对个数：

```text
|R| = |A1|^2 + |A2|^2 + ... + |Ak|^2
```

例：

```text
A = {1,2,3}
划分 {{1,2},{3}}
R = {(1,1),(2,2),(3,3),(1,2),(2,1)}
```

## 八、集合划分与 Bell 数

n 个元素集合上的等价关系个数等于 n 个元素的划分数，即 Bell 数。

常见值：

```text
B1 = 1
B2 = 2
B3 = 5
B4 = 15
```

真题：集合 `{1,2,3,4}` 上有 15 种等价关系。

## 九、偏序关系

偏序关系满足：

```text
自反 + 反对称 + 传递
```

常见偏序：整除关系、包含关系、小于等于关系。

### 整除偏序

在集合 A 上定义 `a R b` 当且仅当 `a | b`。

统计有序对时按每个 a 逐个数：

```text
a=1：能整除所有元素
a=2：能整除 2,4,6,...
...
```

### 极大值与极小值

极大值：没有别的元素比它大，即不存在 `x != a` 且 `a R x`。

极小值：没有别的元素比它小，即不存在 `x != a` 且 `x R a`。

最大值：比所有元素都大。最小值：比所有元素都小。

注意：极大值可以有多个，最大值最多一个；极小值可以有多个，最小值最多一个。

### 链与反链

链：任意两个元素都可比。

反链：任意两个不同元素都不可比。

在整除偏序中：

```text
1 -> 2 -> 4 -> 8
```

是一条链。

像 `{6,7,8,9,10}` 中元素两两互不整除时，可作为反链候选。

## 十、哈斯图

画哈斯图步骤：

1. 先列出集合元素。
2. 按偏序大小分层。
3. 只画覆盖关系，去掉自环和传递边。
4. 小元素在下，大元素在上。

整除关系中，若 `a | b` 且不存在 c 使 `a | c | b`，则 a 到 b 画一条覆盖边。

## 十一、关系复合、逆关系和幂

逆关系：

```text
R^-1 = {(y,x) | (x,y) in R}
```

关系复合：

```text
S o R = {(x,z) | exists y ((x,y) in R and (y,z) in S)}
```

看路径时是：

```text
x --R--> y --S--> z
```

关系幂：

```text
R^0 = I_A
R^1 = R
R^(n+1) = R^n o R
```

传递闭包 `t(R)` 是包含 R 的最小传递关系。若有链 `(1,2),(2,3),(3,4)`，则要补 `(1,3),(2,4),(1,4)`。

例：

```text
R = {(1,2),(2,3),(3,4)}
t(R) = {(1,2),(2,3),(3,4),(1,3),(2,4),(1,4)}
```

共有 6 个有序对。
