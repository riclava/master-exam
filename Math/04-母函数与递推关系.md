# 母函数与递推关系

这份文档覆盖普通母函数、指数型母函数、二项式展开、奇偶次数限制、n 位数计数和常系数递推关系。

## 一、什么时候用母函数

普通母函数：

```text
G(x) = sum a_n x^n
```

常用于相同对象分配、普通序列、整数解和组合数。

指数型母函数：

```text
G_e(x) = sum a_n x^n/n!
```

常用于带排列位置的计数，比如用若干数字组成 n 位数，某些数字出现偶数次/奇数次/至少一次。

## 二、常用展开式

基本几何级数：

```text
1/(1-x) = 1 + x + x^2 + ...
```

广义二项式：

```text
1/(1-x)^m = sum_{k>=0} C(m+k-1,k) x^k
```

例：

```text
1/(1-3t)^2 = sum_{k>=0} C(k+1,k) 3^k t^k
```

`t^3` 的系数：

```text
C(4,3) * 3^3 = 4 * 27 = 108
```

指数函数：

```text
e^x = sum_{k>=0} x^k/k!
```

偶数次：

```text
1 + x^2/2! + x^4/4! + ... = (e^x + e^(-x))/2
```

奇数次：

```text
x/1! + x^3/3! + x^5/5! + ... = (e^x - e^(-x))/2
```

至少一次：

```text
x/1! + x^2/2! + ... = e^x - 1
```

不限制：

```text
1 + x/1! + x^2/2! + ... = e^x
```

## 三、普通母函数例题

若：

```text
G(x) = (1+x+x^2+x^3)/(1-x)
```

因为：

```text
1/(1-x) = 1 + x + x^2 + x^3 + ...
```

所以：

```text
G(x) = 1 + 2x + 3x^2 + 4x^3 + 4x^4 + ...
```

于是：

```text
a0 = 1
a1 = 2
a2 = 3
an = 4, n >= 3
```

## 四、指数型母函数例题

### 例 1：2、4、6、8 组成 n 位数

要求：

1. 2 出现奇数次。
2. 6 出现偶数次。
3. 8 至少出现一次。
4. 4 不限制。

指数型母函数：

```text
G_e(x)
= ((e^x - e^(-x))/2)     // 2 奇数次
  ((e^x + e^(-x))/2)     // 6 偶数次
  (e^x - 1)              // 8 至少一次
  e^x                    // 4 不限制
```

化简后读出 `x^n/n!` 的系数，即为 `a_n`。

### 例 2：1、4、5、8、9 构成 n 位数

要求 4、8 出现偶数次；1、5、9 不限制。

```text
G_e(x) = e^(3x) * ((e^x + e^(-x))/2)^2
       = 1/4 * (e^x + 2e^(3x) + e^(5x))
```

所以：

```text
a_n = 1/4 * (1 + 2*3^n + 5^n)
```

### 例 3：2、3、6、7、9 构成 n 位数

要求 3、7 出现偶数次；2、6、9 不限制。

同理：

```text
G_e(x) = e^(3x) * ((e^x + e^(-x))/2)^2
```

所以：

```text
a_n = 1/4 * (1 + 2*3^n + 5^n)
```

## 五、二项式与多项式系数

二项式：

```text
(1 + ax)^n 中 x^k 的系数 = C(n,k) a^k
```

多项式：

```text
(a + b + c)^n 中 a^r b^s c^t 的系数 = n!/(r!s!t!)
```

带常数系数时，把常数也乘进去。

例：

```text
(3x1 - 5x2 + 8x3)^4 中 x1^2 x2 x3 的系数
= 4!/(2!1!1!) * 3^2 * (-5) * 8
= -4320
```

## 六、递推关系通法

### 一阶线性非齐次

形式：

```text
a_n = c a_(n-1) + d
```

步骤：

1. 齐次部分：`a_n = c a_(n-1)`，通解 `A c^n`。
2. 若 d 是常数，设特解为常数 K；若 `c=1`，常数特解不适用，设特解为 `Bn`。
3. 代入求特解。
4. 用初值求 A。

例：

```text
a_n = a_(n-1) + 2, a_1 = 1
```

可直接累加：

```text
a_n = 2n - 1
```

例：

```text
a_n = 2a_(n-1) - 7, a_0 = 5
```

齐次通解 `A 2^n`，设常数特解 K：

```text
K = 2K - 7 -> K = 7
a_n = A 2^n + 7
a_0 = 5 -> A = -2
a_n = 7 - 2^(n+1)
```

因此：

```text
a_4 = 7 - 32 = -25
```

### 高阶常系数齐次递推

形式：

```text
h_n = h_(n-1) + 9h_(n-2) - 9h_(n-3)
```

写特征方程：

```text
x^3 - x^2 - 9x + 9 = 0
```

若根为 `1, 3, -3`，通解：

```text
h_n = A*1^n + B*3^n + C*(-3)^n
```

再用初值求 A、B、C。

## 七、母函数与递推的考试策略

看到“出现偶数次/奇数次/至少一次/不限制”的 n 位数，优先用指数型母函数。

看到“展开后某项系数”，优先用二项式/多项式系数公式。

看到“普通序列母函数 G(x)”，把有理式拆成已知级数再读系数。

看到“递推关系”，先判断是一阶还是高阶；一阶可累加或特征法，高阶写特征方程。

母函数大题允许中间化简不完全，但必须写出对应的母函数和最终系数表达式。
