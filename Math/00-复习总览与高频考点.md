# 数学基础历年真题复习总览

资料范围：`第1轮复习-全部历年真题-答案解析/数学基础` 下 2012、2013、2014、2018、2019、2020、2021、2022、2023、2024 真题回忆版、2025 真题回忆版，以及 2025 模拟卷，共 12 份 PDF。

## 一、考试最爱考什么

数学基础这部分从历年题看，本质上主要考离散数学。命题不是追求复杂推导，而是反复考固定模型：逻辑符号化、集合与关系、组合计数、母函数/递推、图论与树、证明题。

必须优先掌握五大块：

1. 命题逻辑与谓词逻辑：自然语言符号化、全称量词/存在量词转换、唯一存在、联结词完备集、与非/或非表示、主析取/主合取范式、简单推理证明。
2. 集合、函数与关系：幂集、集合运算、笛卡尔积、二元关系个数、关系性质、等价关系、划分与商集、偏序关系、哈斯图、传递闭包、关系复合。
3. 排列组合与容斥：多重集合排列、圆排列、插空法、捆绑法、错排、容斥原理、隔板法、函数计数、组合恒等式证明。
4. 母函数与递推关系：二项式/多项式系数、普通母函数、指数型母函数、广义二项式、常系数递推关系。
5. 图论与树：握手定理、树的边数和叶子数、生成树、邻接矩阵、二部图、着色数、边着色数、平面图欧拉公式。

## 三、必须会背的“基础公式”

幂集：若 `|A| = n`，则 `|P(A)| = 2^n`。若 A 上二元关系是 `A x A` 的子集，则 A 上二元关系个数为 `2^(n^2)`。

奇偶子集：n 个元素的集合中，奇数个元素子集和偶数个元素子集各有 `2^(n-1)` 个。

排列：`P(n,m) = n!/(n-m)!`。组合：`C(n,m) = n!/(m!(n-m)!)`。

非负整数解：`x1 + ... + xk = n, xi >= 0` 的解数为 `C(n+k-1, k-1)`。

正整数解：`x1 + ... + xk = n, xi >= 1` 的解数为 `C(n-1, k-1)`。

多重集合排列：n 个元素中有重复数 `n1,n2,...,nk`，排列数为 `n!/(n1! n2! ... nk!)`。

圆排列：n 个不同元素围圆桌排列，转圈相同，方案数为 `(n-1)!`。

错排：`D_n = n! * sum_{k=0}^n (-1)^k/k!`，常用 `D_4 = 9, D_5 = 44`。

握手定理：无向图所有顶点度数之和为 `2e`，所以奇度顶点个数为偶数。

树：n 个顶点的树有 `n-1` 条边；二叉树中 `n2 = n0 - 1`，即度为 2 的节点数等于叶子数减 1。

平面图欧拉公式：连通平面图 `v - e + f = 2`。

若简单连通平面图每个面的度数都是 3，则 `e = 3v - 6`。若每个面的度数至少为 4，则 `e <= 2v - 4`。

## 四、必须会做的题型

逻辑符号化：先定义谓词，再写量词。看到“不都是”用 `not forall` 或 `exists ... not`；看到“没有两个不同对象满足某关系”用 `forall x forall y ((x != y) -> not R(x,y))` 或 `not exists x exists y (x != y and R(x,y))`。

关系与划分：一个划分 `{A1,A2,...,Ak}` 对应等价关系 `R = A1 x A1 union A2 x A2 union ... union Ak x Ak`，有序对个数为 `|A1|^2 + |A2|^2 + ... + |Ak|^2`。

偏序整除关系：`a R b` 表示 `a | b`，统计有序对时按每个 a 能整除哪些 b 逐个数；极大值是没有更大可比元素，极小值是没有更小可比元素。

组合计数：遇到“至少”“不允许”“不相邻”“至少一个”优先想容斥、补集、插空、捆绑。

母函数：普通母函数用于相同对象/普通序列；指数型母函数用于不同位置排列、出现次数限制的 n 位数/字符串计数。

递推关系：一阶非齐次递推先求齐次通解，再设常数或一次多项式特解；高阶常系数递推写特征方程。

图论计算：树度数题用“总度数 = 2(n-1)”；平面图题用“欧拉公式 + 面度数求和”；生成树小图可用破圈法，大图可用矩阵树定理。

## 五、复习优先级建议

第一优先级：`03-排列组合与容斥.md`、`05-图论与树.md`、`07-公式与解题模板.md`。这三份决定大多数计算题和填空题。

第二优先级：`01-逻辑与证明基础.md`、`02-集合函数与关系.md`。逻辑符号化每年几乎稳定出现，关系/偏序/等价关系也是常客。

第三优先级：`04-母函数与递推关系.md`、`06-证明题模板.md`。这两块通常在解答题或证明题里拉开差距。

最后用 `08-历年考点索引.md` 对照年份复盘：看到某年题目，要能立刻判断是“隔板法、容斥、整除偏序、欧拉公式、指数型母函数、递推特征方程”中的哪一类。
