# 证明题模板

这份文档专门整理数学基础证明题的常用套路：集合恒等式、关系性质、组合恒等式、鸽巢原理、图论证明。

## 一、集合恒等式证明

最稳方法是元素法。

证明 `X = Y`：

1. 先证 `X subset Y`：任取 `x in X`，推出 `x in Y`。
2. 再证 `Y subset X`：任取 `x in Y`，推出 `x in X`。

模板：

```text
任取 x。
若 x in 左边，则根据定义得到若干条件；
由这些条件推出 x in 右边。
故左边 subset 右边。

反过来，若 x in 右边，同理推出 x in 左边。
故右边 subset 左边。
所以左右相等。
```

常用定义：

```text
x in A union B <=> x in A or x in B
x in A intersect B <=> x in A and x in B
x in A - B <=> x in A and x notin B
x in A △ B <=> (x in A and x notin B) or (x in B and x notin A)
```

## 二、集合条件“当且仅当”

证明：

```text
(A intersect B) union C = A intersect (B union C) iff C subset A
```

分两步：

1. 必要性：假设等式成立，证明 `C subset A`。任取 `x in C`，由左边可得 `x in (A intersect B) union C`，等式推出 `x in A intersect (B union C)`，因此 `x in A`。
2. 充分性：假设 `C subset A`，用元素法证明左右互含。

## 三、关系性质证明

等价关系要证：

```text
自反 + 对称 + 传递
```

偏序关系要证：

```text
自反 + 反对称 + 传递
```

### 既是等价关系又是偏序关系

最简单例子：恒等关系。

```text
I_A = {(x,x) | x in A}
```

证明：

1. 自反：任意 x，`(x,x) in I_A`。
2. 对称：若 `(x,y) in I_A`，则 `x=y`，所以 `(y,x) in I_A`。
3. 传递：若 `(x,y),(y,z) in I_A`，则 `x=y=z`，所以 `(x,z) in I_A`。
4. 反对称：若 `(x,y),(y,x) in I_A`，则 `x=y`。

所以它既是等价关系又是偏序关系。

### 由特殊条件证明等价关系

题型：R 是非空集合 A 上关系，满足：

```text
1. R 自反
2. 若 (a,b) in R 且 (a,c) in R，则 (b,c) in R
```

证明 R 是等价关系：

1. 自反已给。
2. 证对称：若 `(a,b) in R`，又由自反有 `(a,a) in R`，代入条件得 `(b,a) in R`。
3. 证传递：若 `(a,b) in R` 且 `(b,c) in R`，由对称得 `(b,a) in R`，再用条件得 `(a,c) in R`。

## 四、证明不是某性质

要证明命题“若 R 对称且传递，则 R 一定自反”是假的，只需反例。

反例：

```text
A = {1,2}
R = empty
```

空关系满足对称和传递，但不满足自反，因为 `(1,1)` 和 `(2,2)` 不在 R 中。

## 五、组合恒等式证明

组合恒等式优先用“同一件事两种数法”。

### `C(n,r) = C(n,n-r)`

从 n 个元素中选 r 个，相当于决定留下 n-r 个。两种选择一一对应。

### `C(n,k) = C(n-1,k) + C(n-1,k-1)`

从 n 个元素中选 k 个，固定一个特殊元素 a。

1. 不选 a：从剩下 n-1 个中选 k 个，有 `C(n-1,k)`。
2. 选 a：从剩下 n-1 个中再选 k-1 个，有 `C(n-1,k-1)`。

相加得到公式。

### `C(2n,n) = sum C(n,k)^2`

设有 n 个蓝球和 n 个红球，从 2n 个球中选 n 个。

按选出 k 个蓝球分类，则红球选 n-k 个：

```text
C(n,k) * C(n,n-k) = C(n,k)^2
```

对 k 从 0 到 n 求和。

## 六、鸽巢原理证明

基本形式：

```text
n+1 个对象放入 n 个盒子，至少有一个盒子放入至少 2 个对象。
```

棋盘涂色模板：

1. 计算一行/一列可能的涂色模式数。
2. 把模式当鸽巢，把行/列当鸽子。
3. 行/列数量大于模式数，因此至少两行/两列涂色相同。

例：9 x 3 棋盘红蓝涂色。

每行有 3 格，每格 2 种颜色，所以每行模式数：

```text
2^3 = 8
```

9 行放入 8 种模式，必有两行相同。

## 七、图论证明

### 奇度顶点个数为偶数

由握手定理：

```text
sum d(v) = 2e
```

右边为偶数。偶度顶点度数之和为偶数，因此奇度顶点度数之和也为偶数。奇数个奇数相加为奇数，所以奇度顶点个数只能为偶数。

### 最小度为 k 的图有长度至少 k 的路

取图中一条最长路：

```text
v0, v1, ..., vt
```

因为路最长，`v0` 的所有邻接点都必须在这条路上，否则可以把路延长，矛盾。

又因为 `deg(v0) >= k`，所以这条最长路至少包含 k 条从 `v0` 出发可到达的边，故长度至少为 k。

### 平面图边数不等式

若简单连通平面图每个面度数至少为 4：

```text
4f <= 2e
```

结合欧拉公式：

```text
v - e + f = 2
```

由 `f <= e/2` 得：

```text
2 = v - e + f <= v - e + e/2 = v - e/2
```

所以：

```text
e <= 2v - 4
```

## 八、证明题答题习惯

不要只写公式，要写“由定义”“由自反性”“由对称性”“由传递性”“由握手定理”“由鸽巢原理”等理由。

关系证明一定明确三条性质分别证明。

集合证明一定双向包含。

组合恒等式证明尽量讲清楚“同一对象的两种计数方法”。

图论证明先写使用的定理，再代入题目条件。
