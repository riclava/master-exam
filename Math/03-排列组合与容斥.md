# 排列组合与容斥

这份文档覆盖排列、组合、圆排列、多重集合排列、插空法、捆绑法、隔板法、容斥、错排、函数计数和组合恒等式。

## 一、基本公式

排列：

```text
P(n,m) = n!/(n-m)!
```

组合：

```text
C(n,m) = n!/(m!(n-m)!)
```

多项式系数：

```text
(x1 + x2 + ... + xr)^n 中 x1^a1 x2^a2 ... xr^ar 的系数
= n!/(a1! a2! ... ar!)
```

其中 `a1 + a2 + ... + ar = n`。

## 二、二项式与多项式系数

二项式：

```text
(1 + 2x)^n 中 x^k 的系数 = C(n,k) 2^k
```

多项式：

```text
(x1 + x2 + x3)^5 中 x1^2 x2^2 x3 的系数
= 5!/(2!2!1!) = 30
```

带系数多项式：

```text
(3x1 - 5x2 + 8x3)^4 中 x1^2 x2 x3 的系数
= 4!/(2!1!1!) * 3^2 * (-5) * 8
= -4320
```

## 三、多重集合排列

若 n 个元素中有重复，重复次数为 `n1,n2,...,nk`：

```text
排列数 = n!/(n1! n2! ... nk!)
```

例：

```text
3 个 a、1 个 b、2 个 c 共 6 个元素
排列数 = 6!/(3!1!2!) = 60
```

多重集合选取后排列要分类讨论。比如从 `{a,a,b,b,b,c,c,d}` 中选 5 个字母排列，要求 b 至少出现两次：

1. b 出现 2 次，剩余 3 个从 `{a,a,c,c,d}` 选。
2. b 出现 3 次，剩余 2 个从 `{a,a,c,c,d}` 选。
3. 每类按重复元素排列公式计算后相加。

## 四、圆排列

n 个不同元素围圆桌，旋转相同：

```text
(n-1)!
```

若甲乙必须相邻：

1. 把甲乙捆成一个整体。
2. 整体与其他人一起做圆排列。
3. 甲乙内部有 `2!` 种。

例：6 个人围圆桌，甲乙相邻：

```text
(5-1)! * 2! = 48
```

注意：2025 模拟卷给出 240 的解析把圆排列当成线性/或处理口径不统一。按标准圆排列，6 人围圆桌甲乙相邻应为 48；若题目或答案体系固定给 240，要警惕其把座位视为固定座位。考试作答时优先看题干是否强调“转圈视为相同”。

## 五、插空法

用于“不相邻”问题。

例：5 个男同学和 3 个女同学站一排，要求没有 2 个女同学相邻。

1. 先排男生：`5!`。
2. 5 个男生形成 6 个空位。
3. 选 3 个空位放女生并排列：`P(6,3)`。
4. 总数：`5! * P(6,3)`。

圆桌版：5 位老师和 3 位学生围圆桌，学生两两不相邻。

1. 先排 5 位老师成圆：`(5-1)!`。
2. 5 个老师之间有 5 个圆桌间隔。
3. 选 3 个间隔放学生并排列：`P(5,3)`。
4. 总数：`4! * P(5,3) = 1440`。

## 六、交替排列

m 个男生和 m 个女生排成一排，男女相间且女生在最后。

位置固定为：

```text
奇数位：男生
偶数位：女生
```

方案数：

```text
m! * m! = (m!)^2
```

若 5 个文科生和 5 个理科生交替排列，不指定最后性别：

```text
2 * 5! * 5!
```

## 七、捆绑法

用于“必须相邻”问题。

例：四对夫妻围圆桌就座，夫妻都相邻。

1. 把每对夫妻看成整体，4 个整体围圆桌：`(4-1)!`。
2. 每对夫妻内部左右可换：`2^4`。
3. 总数：`3! * 2^4`。

“至少一对夫妻不相邻”可用补集：

```text
全部圆排列 - 每对夫妻都相邻
```

## 八、隔板法

非负整数解：

```text
x1 + x2 + ... + xk = n, xi >= 0
解数 = C(n+k-1, k-1)
```

正整数解：

```text
x1 + x2 + ... + xk = n, xi >= 1
解数 = C(n-1, k-1)
```

例：

```text
x1 + x2 + ... + x6 = 7
非负整数解 = C(12,5) = 792
正整数解 = C(6,5) = 6
```

带下界时先换元：

```text
xi >= ai
令 yi = xi - ai
```

## 九、相同球/不同盒、不同球/相同盒

相同球放入不同盒：隔板法。

不同球放入不同盒且不空：可用容斥或按分配类型分类。

例：4 个相异球放到 3 个相异盒，盒不空。

分配类型只能是 `(2,1,1)`：

1. 选 2 个球为一组：`C(4,2)`。
2. 三组分给 3 个盒：`3!`。
3. 总数：`C(4,2)*3! = 36`。

不同球放入相同盒且不空：按分组，不考虑盒子排列。

例：6 个不同口罩放到 5 个相同盒子，不空。

只能有一个盒子放 2 个，其余各 1 个：

```text
C(6,2) = 15
```

## 十、容斥原理

两个限制：

```text
不出现 A 且不出现 B
= 总数 - |A| - |B| + |A intersect B|
```

例：`a,b,c,d,e,f` 全排列不允许出现 `abc` 和 `ef`。

总数：`6! = 720`。

出现 `abc`：把 `abc` 看整体，与 `d,e,f` 排列，`4! = 24`。

出现 `ef`：把 `ef` 看整体，与 `a,b,c,d` 排列，`5! = 120`。

同时出现：`abc`、`ef`、`d` 三个整体，`3! = 6`。

答案：

```text
720 - 24 - 120 + 6 = 582
```

## 十一、错排

错排 `D_n`：n 个元素都不在原位置。

公式：

```text
D_n = n! * (1 - 1/1! + 1/2! - ... + (-1)^n/n!)
```

常用值：

```text
D_3 = 2
D_4 = 9
D_5 = 44
```

5 个人随机拿伞，全部拿错：`D_5 = 44`。

至少一人拿对：

```text
5! - D_5 = 120 - 44 = 76
```

概率：

```text
76/120 = 19/30
```

## 十二、函数计数和分配

每本书从 5 位专家中选 2 位评审：

```text
C(5,2) = 10
```

5 本不同的书独立选择：

```text
10^5 = 100000
```

注意源解析中有时写成 `10000`，按数学计算应为 `10^5 = 100000`。如果考试中题干是 4 本书则为 `10^4 = 10000`。

从 8 名运动员中选 6 人，再分成 3 个无序二人组：

```text
C(8,6) * (6!/(2!2!2!3!)) = 28 * 15 = 420
```

## 十三、组合恒等式证明

常用组合意义证明。

`C(n,r) = C(n,n-r)`：

从 n 个元素中选 r 个，等价于留下 n-r 个。

`C(n,k) = C(n-1,k) + C(n-1,k-1)`：

按是否包含某个特定元素分类。

`C(2n,n) = sum_{k=0}^n C(n,k)^2`：

把 n 个蓝球和 n 个红球中选 n 个。若选 k 个蓝球，则选 n-k 个红球，方案数 `C(n,k)C(n,n-k)=C(n,k)^2`，对 k 求和。
