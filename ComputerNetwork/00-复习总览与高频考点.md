# 计算机网络历年真题复习总览

资料范围：`第1轮复习-全部历年真题-答案解析/计算机网络` 下 2012、2013、2014、2018、2019、2020、2021、2022、2023、2024、2025 真题回忆版，以及 2025 模拟卷，共 12 份 PDF。

## 一、考试最爱考什么

从历年题目看，计算机网络不是平均用力。必须优先掌握三大块：

1. 网络层：子网划分、CIDR、路由汇聚、最长前缀匹配、IP 分片、RIP/OSPF/BGP、IPv6、NAT。几乎每年都考，且常出计算题。
2. 传输层：TCP 序号/确认号、可靠传输、滑动窗口、流量控制、拥塞控制、UDP 特点、端口/套接字。计算题高频出 TCP 拥塞窗口变化和最大数据率。
3. 数据链路层：以太网、交换机自学习、冲突域/广播域、CSMA/CD、CSMA/CA、VLAN、滑动窗口 ARQ、MTU、CRC/海明码、STP。选择题和填空题高频出现。

物理层、应用层、网络安全分值相对少，但常以“送分题 + 小计算 + 概念辨析”出现，不能丢。

## 二、各层高频程度

| 层次 | 高频程度 | 常见题型 | 必须掌握到什么程度 |
|---|---:|---|---|
| 网络层 | 最高 | 填空、选择、名词、计算 | 会算子网、广播地址、地址范围、路由汇聚、最长前缀匹配、IP 分片；能区分 RIP/OSPF/BGP |
| 传输层 | 最高 | 填空、选择、名词、计算 | 会算 TCP 序号/ACK、窗口变化、拥塞控制曲线、最大数据率；能说明 UDP/TCP 差异 |
| 数据链路层 | 很高 | 填空、选择、名词、计算 | 会算 CSMA/CD 最短帧长、争用期、滑动窗口序号位；理解交换机、VLAN、CSMA/CA |
| 应用层 | 中 | 填空、选择、名词 | 记住 DNS、HTTP、FTP、SMTP/POP3/IMAP 的用途、端口与下层协议 |
| 网络安全 | 中 | 选择、名词 | 区分加密、数字签名、认证、防火墙、VPN、NAT 的作用 |
| 物理层 | 中低 | 填空、计算 | 会算发送时延、传播时延、带宽、奈奎斯特、链路利用率 |
| 体系结构 | 中低 | 填空、选择、名词 | 掌握 OSI/TCP-IP、协议三要素、PDU、SAP、封装效率 |

## 三、必须会背的“关键词”

协议三要素：语法、语义、同步。语法规定格式，语义规定含义和动作，同步规定事件顺序。

各层 PDU：物理层是比特流，数据链路层是帧，网络层是分组/数据报，传输层是报文段或用户数据报，应用层是报文。

地址与标识：主机用 IP 地址标识，数据链路层用 MAC 地址转发，应用进程用端口号标识，套接字是 IP 地址 + 端口号。

RIP：内部网关协议，距离向量算法，跳数为度量，最大有效跳数 15，16 表示不可达，周期性交换路由表。

OSPF：内部网关协议，链路状态算法，泛洪 LSA，同步 LSDB，用 Dijkstra 算法算最短路径，支持分层区域和 ABR。

BGP：外部网关协议，用于自治系统之间，路径向量算法，重视策略，不是单纯最短路径；支持 CIDR。

TCP：面向连接、可靠、字节流、全双工；用序号、确认、重传、校验和、滑动窗口、流量控制、拥塞控制实现可靠传输。

UDP：无连接、不可靠、面向报文、首部简单、时延小；适合 DNS、实时音视频、简单请求响应、广播/组播类业务。

VLAN：逻辑划分广播域，常由 IEEE 802.1Q 标签实现；一个 VLAN 可跨交换机，不同 VLAN 通信需三层设备。

CSMA/CD：有线共享以太网冲突检测；争用期是端到端往返传播时延；最小帧长 = 数据率 × 争用期。

CSMA/CA：无线局域网冲突避免；用 RTS/CTS 缓解隐蔽站问题，用 ACK 确认帧可靠到达。

CIDR：无分类域间路由，写作 `IP/前缀长度`；核心能力是按前缀算地址范围、子网掩码和路由聚合。

MTU：链路层一次帧能承载的最大 IP 数据报长度；IP 数据报超过 MTU 要分片，IPv4 可由路由器分片，重组在目的主机。

## 四、必须会算的题型

子网与 CIDR：给 IP 和掩码，算网络地址、广播地址、可用地址范围、是否同一子网、需要多少子网位/主机位。

路由汇聚：把多个网络地址写成二进制，找最长共同前缀，得到聚合前缀；注意聚合块要能覆盖全部地址。

最长前缀匹配：目的 IP 与每条路由掩码按位与，匹配多个时选前缀最长的；没有匹配走默认路由。

IP 分片：每片总长不超过 MTU；每片数据部分一般取 `MTU - IP首部` 后再向下取 8 的倍数；片偏移单位是 8 字节。

TCP 序号和确认号：序号表示本段第一个字节编号；确认号表示期望收到的下一个字节编号；ACK = 已连续收到的最后字节 + 1。

TCP 拥塞控制：慢开始指数增长，拥塞避免线性增长；超时后 `ssthresh = cwnd/2, cwnd = 1 MSS`；3 个重复 ACK 后快重传/快恢复，通常阈值减半并进入拥塞避免。

滑动窗口序号位：GBN 发送窗口最大 `2^n - 1`；SR 发送窗口和接收窗口最大通常为 `2^(n-1)`。

CSMA/CD：最短帧长 = 数据率 × 2 × 单程传播时延；二进制指数退避第 k 次冲突在 `[0, 2^k - 1]` 个争用期中随机选，截断后 k 不再继续增大。

链路时延与利用率：发送时延 `L/R`；传播时延 `d/v`；RTT 是往返传播时延；停止等待近似利用率 `发送时延 / (发送时延 + RTT)`。

## 五、复习优先级建议

第一优先级：`03-网络层.md`、`04-传输层.md`、`06-计算题模板与公式.md`。这三份决定大部分分数。

第二优先级：`02-数据链路层.md`。选择题、填空题、计算题都有稳定考点，尤其是以太网和滑动窗口。

第三优先级：`01-体系结构与物理层.md`、`05-应用层与网络安全.md`。这些题常不难，但容易因为概念混淆丢分。

最后用 `07-历年考点索引.md` 对照年份回忆：看到某年题目，要能立刻判断属于哪个专题、用哪个公式或概念。
